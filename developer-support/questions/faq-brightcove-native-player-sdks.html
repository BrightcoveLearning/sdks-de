Unten finden Sie eine Liste mit häufig gestellten Fragen zu den Brightcove Native Player SDKs. Haben Sie eine Frage, die Sie hier nicht sehen? Lass uns wissen!

Allgemeine Fragen

Werden Untertitel unterstützt?

Die Player-SDKs unterstützen In-Manifest-WebVTT(HLS) sowie Sidecar-WebVTT und 608-über-708-Untertitel. Weitere Informationen zum Support finden Sie im Videoformate und Untertitel dokumentieren.


Kann ich die Suchleiste anpassen?

Jawohl. Wenn Sie die Brightcove Native Player-Steuerelemente verwenden, können Sie die Elemente anpassen, einschließlich der Suchleiste. Weitere Informationen finden Sie im Native Player-Steuerelemente von Brightcove dokumentieren.


Kann ich das Video mit benutzerdefinierten Steuerelementen überlagern?

Die Player-SDKs zwingen Sie nicht, ihre Steuerelemente zu verwenden. Sie können wählen, ob Sie sie verwenden möchten oder nicht. Wenn Sie die Brightcove Native Player-Steuerelemente verwenden, können Sie die Elemente anpassen. Weitere Informationen finden Sie im Native Player-Steuerelemente von Brightcove dokumentieren.


Kann ich 10sec ff/rew Videofunktionalität hinzufügen?

Die Player-SDKs unterstützen das Suchen, sodass Sie diese Funktionalität implementieren können.


Wird Chromecast unterstützt?

Die Brightcove Native SDKs unterstützen das Streamen von Videos auf mit Chromecast verbundene Fernseher.

  • Verwenden Sie für Android-Geräte die Cast-Plugin für das Native SDK für Android.
  • Verwenden Sie für iOS-Geräte die Cast-Plug-in für Native SDK für iOS.

Werden Wiedergabelisten unterstützt?

Ja, die Brightcove Native SDKs bieten Methoden zum Abrufen eines Videos oder einer Playlist von der Playback API. Einzelheiten finden Sie unter:

  • VideoListSampleApp für das native SDK für Android
  • Abrufen von Brightcove-Assets mit dem Wiedergabedienst für das native SDK für iOS

Android-spezifisch

Wie kann ich HLS auf Android zum Laufen bringen?

Das Player SDK von Brightcove für Android bietet HLS-Wiedergabefunktionen für Android 4.1 und höher.


Was ist mit HLSe?

Das Player SDK von Brightcove für Android unterstützt die Wiedergabe mit HLSe.

iOS-spezifisch

Wird AirPlay unterstützt?

Das Player SDK für iOS unterstützt AirPlay, solange Sie keine Werbung haben. Apples AirPlay ermöglicht es Ihnen, Inhalte von Ihren Apple-Geräten drahtlos zu streamen.


Wird Apple TV unterstützt?

Die neueste Version des Player SDK für iOS unterstützt das tvOS-Betriebssystem von Apple TV.
