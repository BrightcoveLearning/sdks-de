Anforderungen für den App Store

Um die neuen Funktionen des iOS SDK nutzen zu können, verlangt Apple, dass alle neuen Apps, die an den App Store übermittelt werden, mit dem iOS 12.1 SDK oder höher erstellt werden müssen. Diese Anforderung tritt in Kraft ab März 2019.

Das bedeutet, dass Sie zum Einreichen neuer Apps Xcode 10.1 verwenden müssen, das die Tools zum Erstellen von Apps für die neuesten SDKs bereitstellt, einschließlich iOS 12, tvOS 12 und Swift 4.2.

Einzelheiten finden Sie im Senden von iOS-Apps an den App Store Dokumentation.

Native SDK-Unterstützung

Dem Beispiel von Apple folgend, unterstützt das Brightcove Native SDK für iOS/tvOS v6.3.10+ das Erstellen mit Xcode 10.1. Falls noch nicht geschehen, migrieren Sie bitte so schnell wie möglich auf Xcode 10.1.

Das Brightcove Native SDK für iOS/tvOS v6.3.5+ unterstützt das Erstellen mit Xcode 8 nicht mehr.
