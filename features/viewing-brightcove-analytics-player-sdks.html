Überblick

Im Video Cloud Analytics-Modul können Sie Metrikdaten auf Kontoebene oder auf Detailebene, z. B. per Video, anzeigen. Ausführliche Informationen zum Anzeigen der Leistungsberichte in Video Cloud Analytics finden Sie im Referenz zum Leistungsbericht dokumentieren.

Hier sind einige Optionen zum Anzeigen Ihrer Daten:

  • Konto-ID
  • Ziel
  • Spieler
  • Quelle
Brightcove Analytics konvertiert die von Ihnen festgelegten Ziel- und Quellwerte in allen Berichten in einen ausschließlich in Kleinbuchstaben geschriebenen Zeichenfolgenwert.

Sie können auch lernen über Senden von Daten an Video Cloud Analytics.

Konto-ID

Zeigen Sie die Quellwerte an, die Sie in Ihrer App festgelegt haben.

accountId - Videoebene

Sehen Sie sich die Videoanalyse an, in der Sie die accountId festgelegt haben.

  1. Wählen Sie im Analytics-Modul Leistung im Menü auf der linken Seite und wählen Sie Video aus dem angezeigten Kontextmenü.

  2. Die Andere/Dritte Listenelement zeigt eine Zusammenfassung der Leistung aller Videos an, für die Sie die Eigenschaft accountId festgelegt haben. Diese Videos werden nicht separat aufgeführt, da Brightcove Analytics derzeit keine detaillierte Berichterstellung für Videos unterstützt, die nicht über den Brightcove-Katalogdienst abgerufen werden.

Ziel

Zeigen Sie die Zielwerte an, die Sie in Ihrer App festgelegt haben.

Ziel - Kontoebene

Zeigen Sie die Zielwerte an, die Sie auf Kontoebene festgelegt haben.

  1. Wählen Sie im Analytics-Modul Leistung im Menü auf der linken Seite und wählen Sie Domain anzeigen aus dem angezeigten Kontextmenü.

  2. Dieser Bericht zur Kontoleistung listet Zielwerte auf, einschließlich der Werte, die Sie in Ihrem App-Code angegeben haben.

  3. Wählen Sie eine Zieldomäne aus, um die Details anzuzeigen.

Ziel - Videoebene

Zeigen Sie die Zielwerte an, die Sie auf Videoebene festgelegt haben.

  1. Wählen Sie im Analytics-Modul Leistung im Menü auf der linken Seite und wählen Sie Video aus dem angezeigten Kontextmenü.

  2. Wählen Sie ein Video zum Anzeigen aus.

  3. Erweitere die Leistung von Dropdown-Menü und wählen Sie das Domain anzeigen Möglichkeit.

  4. In diesem Bericht zur Videoleistung werden Zielwerte aufgeführt, einschließlich der Werte, die Sie in Ihrem App-Code angegeben haben.

Spieler

Zeigen Sie die Spieler auf Kontoebene an, die in Ihrer App verwendet werden.

Spieler - Kontoebene

Zeigen Sie Spieleranalysen für Spieler in Ihrer App an.

  1. Wählen Sie im Analytics-Modul Leistung im Menü auf der linken Seite und wählen Sie Spieler aus dem angezeigten Kontextmenü.

Quelle

Zeigen Sie die Quellwerte an, die Sie in Ihrer App festgelegt haben.

Quelle - Kontoebene

Zeigen Sie die Quellwerte an, die Sie auf Kontoebene festgelegt haben.

  1. Wählen Sie im Analytics-Modul Leistung im Menü auf der linken Seite und wählen Sie Traffic-Quelle aus dem angezeigten Kontextmenü.

  2. In dem Zusammenfassung Liste, wählen Sie die Verweisung Link, um einen detaillierten Bericht nach Quelle anzuzeigen.

  3. Dieser Bericht zur Kontoleistung enthält die Quellwerte, die Sie in Ihrem App-Code angegeben haben.

Quelle - Videopegel

Zeigen Sie die Quellwerte an, die Sie auf Videoebene festgelegt haben.

  1. Wählen Sie im Analytics-Modul Leistung im Menü auf der linken Seite und wählen Sie Video aus dem angezeigten Kontextmenü.

  2. Wählen Sie ein Video zum Anzeigen aus.

  3. Erweitere die Leistung von Dropdown-Menü und wählen Sie das Traffic-Quelle Möglichkeit.

  4. Diese Liste zeigt Ihnen die Leistung für Quellen, einschließlich der Quellwerte, die Sie in Ihrem App-Code angegeben haben. Beachten Sie, dass das Analytics-Modul kein Drilling in jedes Listenelement unterstützt, wie z Direkte.

Senden von Daten an Video Cloud

Ausführliche Informationen zum Senden von Daten aus Ihren Apps an Video Cloud Analytics finden Sie unter:

  • Brightcove Player-SDK für iOS
  • Brightcove Player-SDK für Android

Wenn Sie den im SDK enthaltenen Brightcove-Player und -Katalog zum Abrufen und Wiedergeben von Videos verwenden, werden automatisch Analysedaten erfasst und im Video Cloud-Analysemodul angezeigt.
