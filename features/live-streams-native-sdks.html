Einführung

Die Livestream-Unterstützung mit den Brightcove Native Player SDKs bedeutet, dass Sie schnell und einfach hochwertige Liveübertragungen aus jeder Quelle in Ihre Anwendung integrieren können.

Die nativen SDKs unterstützen sowohl Live-Streaming als auch DVR-Live-Streaming. Beim Live-Streaming unterstützen die Native SDKs zeitgesteuerte Metadaten/ID3-Tags.

Das Android SDK unterstützt ID3-Metadaten-Tags bei Verwendung der BrightcoveExoPlayer-Ansicht.

Eine Möglichkeit, Live-Streams zu erstellen, ist die Verwendung von Brightcove Live.

Eine vollständige Liste der Optionen für die Bereitstellung von Livestreams finden Sie unter Bereitstellung von Livestreams dokumentieren.

Brightcove Live

Brightcove Live ist der einfachste Weg, um Ihre Livestreams zu erstellen. Damit können Sie hochauflösende (HD) Live-Videostreams mit adaptiver Bitrate bereitstellen und monetarisieren, die sowohl skalierbar als auch kostengünstig sind.

Richten Sie Ihre Brightcove Live-Streams ein und spielen Sie sie mit den Native Player SDKs für Android, iOS und tvOS nahtlos auf Mobilgeräten und einem angeschlossenen Fernseher ab. Erfahren Sie mehr über Brightcove Live mit den folgenden Informationen:

  • Brightcove Live
  • Schnellstart: Brightcove Live-API

Unterstützte Funktionen

Das Live-Erlebnis wird durch die folgenden Funktionen verbessert, die von den Native SDKs unterstützt werden:

Vorkonfigurierte Cloud-DVR-Steuerung

DVR Live geht über den klassischen Ansatz des Live-Streamings hinaus, indem es den Zuschauern ermöglicht, die Kontrolle über ihr Seherlebnis zu übernehmen. DVR Live bietet dem Zuschauer die volle Kontrolle über die Wiedergabe und ermöglicht es Ihrem Publikum, vergangene Momente in einer VOD-artigen Umgebung zu suchen. Springen Sie dann mit einem Klick auf eine Schaltfläche nahtlos zur Live-Action zurück, um die Wiedergabe in Echtzeit fortzusetzen.

Serverseitige Anzeigeneinfügung von Brightcove Live (SSAI)

Mit der serverseitigen Anzeigeneinfügung (SSAI) können Sie Anzeigen serverseitig in Ihren Live-Stream einfügen und dabei alle Werbeblocker umgehen. Weitere Informationen zum Einrichten von SSAI mit Ihren Livestreams finden Sie im Brightcove Live-API: Serverseitige Anzeigenschaltung dokumentieren.
