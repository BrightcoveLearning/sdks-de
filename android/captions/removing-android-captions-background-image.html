Überblick

Wenn Sie ein Video mit Untertiteln in eine mit dem Brightcove Native Player SDK für Android erstellte App laden, wird im Mediencontroller eine Schaltfläche für Untertitel angezeigt.

Wählen Sie im Mediencontroller die Schaltfläche Untertitel, um einen Dialog mit a . anzuzeigen Einstellungen Taste.

Wähle aus Einstellungen , um ein dynamisches Hintergrundbild anzuzeigen, um Untertitel anzuzeigen, während Sie sie konfigurieren.

Die Hintergrundbilddatei der Untertitel hat eine Größe von 400 KB. Wenn Sie dieses Bild entfernen möchten, können Sie es durch eine leere Bilddatei mit demselben Namen ersetzen.

Hintergrundbild entfernen

Führen Sie diese Schritte aus, um das standardmäßige Hintergrundbild für Untertitel zu entfernen:

  1. In dem Brightcove Natives SDK für Android , navigieren Sie zum res/drawable-nodpi/caption_background.jpg Datei.
  2. Erstellen Sie eine leere Datei mit demselben Namen, caption_background.jpg.
  3. Ersetzen Sie das Vorhandene caption_background.jpg Datei mit Ihrer leeren Datei mit dem gleichen Namen.
  4. Kompilieren Sie Ihr Projekt neu.
