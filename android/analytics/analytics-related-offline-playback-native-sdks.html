Angesichts der steigenden Nachfrage, Videoinhalte auf Mobilgeräten herunterzuladen und ohne Verbindung anzusehen, kann die Fähigkeit, das Sehverhalten der Benutzer zu verstehen, eine Herausforderung darstellen.

Mit den Brightcove Native Player SDKs für Android und iOS können Sie das Benutzerverhalten verfolgen und lernen, indem Sie Analysen zur Offline-Wiedergabe bereitstellen.

Überblick

Die mit der Offline-Wiedergabe verknüpften Analysedaten umfassen sowohl Messwerte zum Video-Download als auch zur Offline-Wiedergabe. Die Native Player SDKs verarbeiten diese Messwerte wie folgt:

  • Downloaddaten werden während des Videodownloads an Brightcove gesendet.
  • Offline-Wiedergabedaten werden lokal auf Ihrem Gerät gespeichert, bis es mit einem Netzwerk verbunden ist.
  • Als App-Entwickler können Sie wählen, ob das SDK Daten mit einer mobilen Verbindung hochlädt oder Uploads nur auf WIFI beschränkt.
  • Die Standardobergrenze für die gespeicherte Datenmenge ist auf 500 MB festgelegt, aber der App-Entwickler kann diese Einstellung überschreiben. Dies entspricht mehreren hundert Stunden Offline-Wiedergabemetriken.
  • Für alle Ereignisse gibt es eine Wiederholungslogik, aber wir priorisieren Videoaufrufe und senden sie zuerst.

Daten zum Video-Download

Zu den Metriken gehören:

  • Download-Anfrage, einschließlich Wiedergabe-Bitrate und Wiedergabe-Dimensionen
  • Download abgeschlossen
  • Download-Fehler
  • Stornierung herunterladen

Analysen anzeigen

Verwendung der Analytics-API , können Sie die folgenden Download-Messwerte anzeigen:

Abmessungen Gebiet
Video video_download_request
  video_download_complete
  video_download_cancellation
  video_download_error

Weitere Informationen zum Anzeigen dieser Download-Messwerte über die Analytics-API finden Sie im Dimension: Video dokumentieren.

Musteranfrage

Um Daten für die downloadspezifischen Felder zurückzugeben, muss Ihr HTTP GET Anfrage an die Analytics-API wird ähnlich aussehen:

https://analytics.api.brightcove.com/v1/data?accounts=Ihre Konto-ID&dimensions=video&fields=video_download_request,video_download_complete,video_download_cancellation,video_download_error

Offline-Wiedergabedaten

Zu den Metriken gehören:

  • Gerätetyp, Hersteller, Betriebssystem, Betriebssystemversion
  • Geographische Lage
  • Domain
  • Video-ID
  • Videoaufrufe (Anzahl der Offline-Videoaufrufe)
  • Videointeraktion (Prozentsatz des angesehenen Videos)
