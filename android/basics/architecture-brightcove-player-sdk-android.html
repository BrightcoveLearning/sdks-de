Überblick

Das Brightcove Native SDK für Android vereinfacht die Integration der Videowiedergabe mit Brightcove Video Cloud und vielen beliebten Werbe-, Analyse- und DRM-Lösungen.

Das Native SDK bietet auch Unterstützung für Untertitel für Pre-KitKat-Geräte und leicht anpassbare Player-Steuerelemente mit ähnlichem Aussehen und Verhalten wie der Brightcove Web Player, um eine konsistente Benutzererfahrung auf allen Plattformen zu bieten.
