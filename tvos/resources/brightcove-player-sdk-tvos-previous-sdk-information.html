    Brightcove Player SDK für tvOS: Vorherige SDK-Informationen

    Dieses Thema zeigt die vorherigen SDK- und Plugin-Versionen für das Native Player SDK für tvOS.
    Die folgenden Links für die Vorgängerversion des Brightcove Native Player SDK for tvOS dienen nur als Referenz. Es wird empfohlen, die neueste Version des SDK zusammen mit den Plugin-Integrationen zu verwenden.

    Frühere Versionen

    Das Brightcove Player SDK for tvOS ist im Player SDK für iOS enthalten. Die folgenden Links gelten für die vorherige Version des Brightcove Native Player SDK für iOS.
