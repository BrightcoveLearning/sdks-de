    Die native Brightcove SDKs

    Willkommen bei der Feature-Level-Dokumentation von Brightcove Native SDKs! Hier finden Sie alles, was Sie über unsere Native SDKs für Mobil- und TV-Geräte wissen müssen.

    Übersicht

    Native-SDKs sind keine Video-Player. Diese SDKs sind Integrationspunkte für Werbung, Analytics und DRM sowie Untertitel.

    In dieser Video-Einführung erhalten Sie einen kurzen Überblick über Native-SDKs:

     

     

    Die Native-Entwicklung kann kompliziert werden, und Sie haben viele Möglichkeiten zur Integration von Video in Ihre Anwendungen. Wenn Sie mit diesem Thema nicht vertraut sind, empfehlen wir Ihnen, sich über die Optionen zur Videowiedergabe in der Native-Umgebung zu informieren:

    Warum sollten Sie die Brightcove Native SDKs verwenden? Lesen Sie die Anleitung zur nativen Videowiedergabe.

    Sehen Sie sich die folgenden Videos an, um zu erfahren, wie Sie mit den Native SDKs einfache Video-Apps erstellen können:

    •  Video-Training: Entwicklung mit den Native SDKs

    Features

    Die Brightcove SDKs unterstützen eine Vielzahl von Funktionen, darunter die folgenden:

    • Untertitel
    • Advertising
    • Analytics
    • Contentschutz
    • Live-Wiedergabe
    • Standardsteuerelemente der Benutzeroberfläche
    • 360°-Video
    • Offline Wiedergabe

    Eine vollständige Liste der Funktionen finden Sie im Dokument Brightcove Native SDK Features .

    Bleiben Sie mit dem Brightcove-Blogüber Informationen auf Funktionsebene auf dem Laufenden.

    Produkte

    Die Brightcove Native SDKs wurden speziell für Entwickler nativer Plattformen entwickelt und zielen darauf ab, die Markteinführungszeit zu minimieren, indem sie die neuesten Standards in der Anwendungsentwicklung in jeder Umgebung einhalten.

    Die Brightcove Native SDKs basieren auf den Frameworks für native Player. Auf diese Weise können Sie die Vorteile der Geschwindigkeit, Leistung und Flexibilität des Native-OS nutzen und für ein umfassendes Video-Erlebnis sorgen, das leicht zu integrieren ist. Außerdem halten sich Native-SDKs an Konventionen für die Speicherverwaltung, Benennung und Designmuster, um einen möglichst leicht Einstieg zu ermöglichen.

    Die Native SDKs sind für die Integration in die folgenden Gerätetypen konzipiert:

    • Mobile-Geräte
    • Verbundenes Fernsehen/Top Boxen einstellen

    Mobilgeräte

    Wir bieten die folgenden SDKs für Mobilgeräte an, die jeweils auf ihren eigenen Zweck spezialisiert sind:

    • Brightcove Native SDK für iOS
    • Brightcove Native SDK für Android

    Brightcove Native SDK für iOS

    Das Brightcove Native SDK für iOS folgt modernen iOS-Konventionen. Egal, ob Sie sich in eine etablierte Objective-C-App oder eine brandneue Swift-App integrieren, Sie sollten sich wie zu Hause fühlen.

    Unterstützte iOS-Versionen
    Um die neuesten technologischen Verbesserungen von Apple nutzen zu können, bieten wir derzeit Unterstützung für:

    • iOS-14
    • iOS-13
    • ios 12,4,1

    Mit der Veröffentlichung von iOS 9 und Xcode 7 erfahren Sie, wie Sie mit einer neuen Funktion namens App Transport Security (ATS)arbeiten.

    Mit einer Kombination aus intern gesammelten Metriken und den öffentlichen Berichten zur Nutzung von iOS-Geräten von Apple zielt das Brightcove SDK für iOS darauf ab, die überwiegende Mehrheit der Benutzer zu erreichen.

    Wenn Sie ein Entwickler sind, der nach weiteren Details sucht, lesen Sie die technische Dokumentation zum Brightcove Native SDK for iOS .

    Brightcove Native SDK für Android

    Das Brightcove Native SDK für Android zielt darauf ab, die überwiegende Mehrheit der Benutzer auszugleichen und gleichzeitig die Verbesserungen zu nutzen, die in den neuesten Versionen von Android veröffentlicht wurden.

    Unterstützte Android-Versionen
    Um diese Reichweite mit der Möglichkeit in Einklang zu bringen, die von Google veröffentlichten Verbesserungen zu nutzen, bieten wir derzeit Unterstützung für die folgenden Versionen von Android:

    • 10.0
    • 9.0
    • 8,0, 8.1
    • 7.0, 7.1
    • 6.0

    Das Brightcove-SDK für Android nutzt eine Kombination aus intern gesammelten Daten und dem öffentlich zugänglichen Android-Gerätenutzungsdatenbericht von Google, um den größten Teil der Nutzer zu erreichen.

    Wenn Sie ein Entwickler sind, der nach weiteren Details sucht, lesen Sie die technische Dokumentation zu Brightcove Native SDK for Android .

    Verbundenes Fernsehen/Top Boxen einstellen

    Wir bieten die folgenden SDKs zur Integration in TV-Set-Top-Boxen an:

    • Brightcove Native SDK für tvOS
    • Brightcove Native SDK für Android

    Brightcove Native SDK für tvOS

    Fesseln Sie Ihr Publikum im Wohnzimmer mit videozentrierten Anwendungen in der 4. Generation von Apple TV, indem Sie Video Cloud-Inhalte über das Brightcove Native SDK für tvOS nutzen!

    Unterstützte tvOS-Versionen
    Um die neuesten technologischen Verbesserungen von Apple nutzen zu können, bieten wir derzeit Unterstützung für:

    • tvOS 14
    • tvOS 13
    • tvOS 12.4.1

    Erfahren Sie mehr über Connected TV-Apps mit den Native SDKs.

    Für Kunden, die ohne Ausgaben für die Entwicklung ein fesselndes Apple TV-Erlebnis erstellen möchten, bietet Brightcove eine Lösung, die ein UX-Design (User Experience), Video-Wiedergabe in Topqualität, einen leicht zu verwaltenden Content-Katalog und zuverlässige Analytics umfasst. Testen Sie Brightcove Jump Start for Apple TV.

    Wenn Sie ein Entwickler sind, der nach weiteren Details sucht, lesen Sie die technische Dokumentation zu Brightcove Native SDK for tvOS .

    Brightcove Native SDK für Android

    Erweitern Sie die Reichweite Ihrer Videoinhalte auf eine Vielzahl von Android-basierten Smart-TV-Plattformen und Fire TV mit dem Brightcove Native SDK für Android.

    Wir bieten derzeit Unterstützung für:

    • Android TV
    • Fire TV (FireOS 5.0+ Geräte)

    Erfahren Sie mehr über Connected TV-Apps mit den Native SDKs.

    Wenn Sie ein Entwickler sind, der nach weiteren Details sucht, lesen Sie die technische Dokumentation zu Brightcove Native SDK for Android .

    Anmelden

    Die Brightcove Native SDKs sind Teil der Brightcove Video Cloud- und Brightcove Player-Pakete! Kontaktieren Sie uns, wenn Sie sich anmelden möchten.
