Überblick

Die Widevine-Quellenauswahlrichtlinie bestimmt, welche Quelle aus einem BCOVVideo-Objekt ausgewählt werden soll.

+ sourceSelectionWidevineWithScheme:

Wählen Sie das erste Widevine-Video mit dem angegebenen Schema aus. Wenn keine Widevine-Quelle mit diesem Schema gefunden wird, geben Sie die erste Widevine-Quelle eines beliebigen Schemas zurück. Wenn keine Widevine-Quelle gefunden wird, greifen Sie auf die HLS-Schema-basierte Quellenauswahlrichtlinie zurück: Wählen Sie das erste Video in HLS mit dem angegebenen Schema aus. Wenn keine solche Quelle gefunden wird, wird die erste HLS-Quelle mit einem beliebigen Schema zurückgegeben. Wenn keine HLS-Quellen vorhanden sind, wird die erste MP4 zurückgegeben. Wenn keines der oben genannten gefunden wird, wird die erste Quelle jeglicher Art zurückgegeben.

+ (BCOVBasicSessionProviderSourceSelectionPolicy)sourceSelectionWidevineWithScheme:(NSString *)planen

Parameter

planen

Das bevorzugte Schema (kBCOVSourceURLSchemeHTTP oder kBCOVSourceURLSchemeHTTPS) der Quell-URL.

Rückgabewert

Eine Quellenauswahlrichtlinie, die HLS mit dem angegebenen Schema bevorzugt.

+ sourceSelectionWidevine

Wählen Sie das erste Widevine-Video aus und bevorzugen Sie HTTPS gegenüber HTTP. Wenn keine Widevine-Quelle gefunden wird, greifen Sie auf die grundlegende Richtlinie zur Quellenauswahl zurück: Wählen Sie das erste Video in HLS aus, unabhängig vom Schema. Wenn keine HLS-Quelle gefunden wird, wählen Sie die erste Quelle mit einem deliveryType von “MP4”. Wenn keiner gefunden wird, wird die erste Quelle einer beliebigen Art zurückgegeben. Dies ist die Standardrichtlinie für die Quellenauswahl.
