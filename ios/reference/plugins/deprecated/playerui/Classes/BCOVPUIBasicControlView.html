+ layoutViewWithControlFromTag:width:elastizität:

Convenience-Factory-Methode zum Erstellen von a Layout Ansicht mit einer Unteransicht, die durch das angegebene Tag bestimmt wird. Layoutansichten werden in Arrays gebündelt und zum Erstellen eines BCOVPUIControlLayout Beispiel.

+ (BCOVPUILayoutView *)layoutViewWithControlFromTag:(BCOVPUIViewTag)Schild Breite: (CGFloat)Breite Elastizität: (CGFloat)Elastizität

Parameter

Schild

Ein Tag-Wert, der das gewünschte Steuerelement angibt oder leer Layout Aussicht.
Breite

Die minimale Breite des neuen Layout Aussicht. Übergeben Sie kBCOVPUILayoutUseDefaultValue, um die Standardbreite für das jeweilige Steuerelement zu verwenden.
Elastizität

Der relative Betrag, den a Layout Die Ansicht kann erweitert werden, um den Raum zu füllen. Elastizität ist relativ zu anderen Layout Ansichten.

Rückgabewert

EIN BCOVPUILayoutView -Instanz initialisiert mit einer Unteransicht, die durch Tag, Breite und Elastizität angegeben wird.

Diskussion
