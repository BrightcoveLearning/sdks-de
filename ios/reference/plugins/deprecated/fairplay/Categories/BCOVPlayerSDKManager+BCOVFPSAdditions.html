BCOVPlayerSDKManager(BCOVFPSAdditions) Kategoriereferenz

Deklariert in BCOVFPSComponent.h

Überblick

Fügt BCOVPlayerSDKManager Methoden hinzu, um FairPlay-Player und Sitzungsanbieter zu erstellen.

– createFairPlayPlaybackControllerWithAuthorizationProxy:

Erstellt einen FairPlay-fähigen Wiedergabe-Controller und gibt ihn zurück. Wenn Sie die dynamische Bereitstellung von Video Cloud verwenden, wird beim Abspielen einer Wiedergabeliste automatisch ein Fairplay-Anwendungszertifikat von BCOVVideo abgerufen. Eine Bewerbungsbescheinigung können Sie nach wie vor separat telefonisch abgeben BCOVBasicPlaybackController addApplicationCertificate:identifier:.

- (nicht null id<BCOVPlaybackController>)createFairPlayPlaybackControllerWithAuthorizationProxy:(nicht null id<BCOVFPSAuthorizationProxy>)Stellvertreter

Parameter

Stellvertreter

Die BCOVFPSAuthorizationProxy für diesen Sitzungsanbieter zu verwenden. Darf nicht Null sein.

Rückgabewert

Ein Playback-Controller.

- (nicht null id<BCOVPlaybackSessionProvider>)createFairPlaySessionProviderWithAuthorizationProxy:(nicht null id<BCOVFPSAuthorizationProxy>)Stellvertreter UpstreamSessionProvider:(nullable id<BCOVPlaybackSessionProvider>)Anbieter

Parameter

Stellvertreter

Die BCOVFPSAuthorizationProxy für diesen Sitzungsanbieter zu verwenden. Darf nicht Null sein.
Anbieter

Und Upstream-Sitzungsanbieter zum Verlinken. Wenn nil, wird ein BCOVBasicSessionProvider verwendet.

– createFairPlayPlaybackControllerWithApplicationCertificate:authorizationProxy:viewStrategy:

Erstellt einen FairPlay-fähigen Wiedergabe-Controller und gibt ihn zurück.

- (nicht null id<BCOVPlaybackController>)createFairPlayPlaybackControllerWithApplicationCertificate:(nullable NSData *)appCert authorizationProxy:(nicht null id<BCOVFPSAuthorizationProxy>)Stellvertreter viewStrategy:(nullable BCOVPlaybackControllerViewStrategy)AnsichtStrategie

Parameter

appCert

Das Fairplay-Anwendungszertifikat für diesen Sitzungsanbieter. Wenn Sie Brightcove FPS Server verwenden, kann dies abgerufen werden durch BCOVFPSBrightcoveAuthProxy. Kann null sein.
Stellvertreter

Die BCOVFPSAuthorizationProxy für diesen Sitzungsanbieter zu verwenden. Darf nicht Null sein.
AnsichtStrategie

Die zu verwendende Ansichtsstrategie.

Rückgabewert

Ein Playback-Controller.

Diskussion

Erstellt einen FairPlay-fähigen Wiedergabe-Controller und gibt ihn zurück.
