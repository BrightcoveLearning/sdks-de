+ videoPropertiesVMAPAdTagUrlAdsRequestPolicy

Gibt eine Richtlinie für Anzeigenanfragen zurück, die das BCOVVideo-Objekt in jeder Wiedergabesitzung auf die anzufordernde VMAP-Anzeigen-Tag-URL überprüft. Wenn das Videoobjekt’s properties enthält einen Eintrag mit dem Schlüssel kBCOVIMAAdTag , wird angenommen, dass der Wert dieses Eintrags ein NSString ist, der die VMAP-Anzeigen-Tag-URL enthält. Diese URL wird verwendet, um eine VMAP-IMAAdsRequest für diese Wiedergabesitzung zu erstellen.
