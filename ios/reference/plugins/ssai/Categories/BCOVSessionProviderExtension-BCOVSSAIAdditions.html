– ssai_seekToTime:completionHandler:

Diese Methode implementiert das Standardsuchverhalten für das BCOVSSAI-Plugin. Diese Methode muss im Hauptthread aufgerufen werden. Der Empfänger ignoriert diese Nachricht, wenn entweder eine Anzeige läuft oder ein vorheriger Anruf an ssai_seekToTime:completionHandler: wird noch bearbeitet. Siehe auch ssai_canSeek.

- (void)ssai_seekToTime:(CMTime)Zeit CompletionHandler:(void ( ^ ) ( BOOL beendet ))FertigstellungHandler

Parameter

Zeit

Der logische Zeitpunkt, um zu suchen
FertigstellungHandler

Der auszuführende Block, wenn die Suche ausgeführt wird

Diskussion

Diese Methode implementiert das Standardsuchverhalten für das BCOVSSAI-Plugin. Diese Methode muss im Hauptthread aufgerufen werden. Der Empfänger ignoriert diese Nachricht, wenn entweder eine Anzeige läuft oder ein vorheriger Anruf an ssai_seekToTime:completionHandler: wird noch bearbeitet. Siehe auch ssai_canSeek.

Die Regeln sind wie folgt:

  • Wenn versucht wird, die Zeit des logischen Inhalts zu beenden, werden die Post-Rolls abgespielt.
  • Wenn Sie über eine Anzeigensequenz suchen, wird die Anzeige abgespielt und der Abspielkopf wird dort fortgesetzt, wo die Suche versucht hatte, zu suchen.
  • Wenn nach mehreren Anzeigensequenzen gesucht wird, wird nur die letzte Anzeigensequenz wiedergegeben.
  • Wenn Sie rückwärts suchen, werden keine Anzeigen abgespielt.
  • Wenn Sie nach bereits abgespielten Anzeigen suchen, werden diese erneut abgespielt.

– ssai_absoluteTimeAtContentTime:

- (CMTime)ssai_absoluteTimeAtContentTime:(CMTime)InhaltZeit

Parameter

InhaltZeit

Ein Zeitversatz in den Videoinhalt.

Rückgabewert

Die absolute Zeit, die der angegebenen Inhaltszeit entspricht.

Diskussion

Gibt die absolute Zeit zurück, die der angegebenen Inhaltszeit entspricht.

Diese Methode wird verwendet, um eine absolute Streamzeit zu erhalten, die einer Zeit entspricht, die relativ zum Inhalt ist. Der Stream kann beispielsweise eine 30-Sekunden-Anzeigensequenz um enthalten kCMTimeZero , danach beginnt der Inhalt. In diesem Fall würde eine Content-Zeit von null einer absoluten Stream-Zeit von dreißig Sekunden entsprechen und eine Content-Zeit von 2:00 einer absoluten Stream-Zeit von 2:30 (vorausgesetzt, dass vor dieser Zeit keine anderen Anzeigensequenzen auftreten). .

– ssai_contentTimeAtAbsoluteTime:

Gibt die Inhaltszeit zurück, die der angegebenen absoluten Zeit entspricht.

- (CMTime)ssai_contentTimeAtAbsoluteTime:(CMTime)absoluteZeit

Parameter

absoluteZeit

Ein Zeitversatz in die Videonutzlast.

Rückgabewert

Die Inhaltszeit, die der angegebenen absoluten Zeit entspricht.

Diskussion

Gibt die Inhaltszeit zurück, die der angegebenen absoluten Zeit entspricht.

Diese Methode wird verwendet, um eine Inhaltsstromzeit zu erhalten, die einer Zeit entspricht, die relativ zur Nutzlast ist. Der Stream kann beispielsweise um 1:30 Uhr eine 30-Sekunden-Anzeigensequenz enthalten, nach der der Inhalt beginnt.

In diesem Fall würde eine Inhaltszeit von Null einer absoluten Streamzeit von Null entsprechen, eine Inhaltszeit von 2:00 würde einer absoluten Streamzeit von 2:30 entsprechen und eine Inhaltszeit von 1:30 würde einer beliebigen entsprechen absolute Streamzeit von 1:30 bis 2:00 Uhr.
