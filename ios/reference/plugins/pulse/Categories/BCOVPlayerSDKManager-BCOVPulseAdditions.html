BCOVPlayerSDKManager(BCOVPulseAdditions) Kategoriereferenz

Deklariert in BCOVPulseComponent.h

Überblick

Kategoriemethoden zum BCOVPlayerSDKManager hinzugefügt, um Pulse zu unterstützen.

– createPulsePlaybackControllerWithPulseHost:contentMetadata:requestSettings:adContainer:companionSlots:viewStrategy:options:

Erstellt und gibt einen neuen Wiedergabecontroller mit dem angegebenen Pulse-Host, dem OOContentMetadata-Objekt, dem OORequestSettings-Objekt und den Benutzeroptionen zurück. Der zurückgegebene Playback-Controller wird mit einem Pulse-Sitzungsanbieter konfiguriert.

- (id<BCOVPlaybackController>)createPulsePlaybackControllerWithPulseHost:(NSString *)pulseHost contentMetadaten:(OOContentMetadaten *)contentMetadaten requestSettings:(OORequestSettings *)AnfrageEinstellungen adContainer:(UIView *)AnzeigenContainer BegleiterSlots:(NSArray *)BegleiterSlots viewStrategy:(BCOVPlaybackControllerViewStrategy)Strategie Optionen:(NSWörterbuch *)Optionen

Parameter

pulseHost

Der Pulse-Hostname für das verwendete Clientkonto.
contentMetadaten

Ein OOContentMetadata, das verwendet wird, um das Sitzungsobjekt zu konfigurieren, das vom zurückgegebenen Wiedergabe-Controller verwendet wird.
AnfrageEinstellungen

Ein OORequestSettings, das verwendet wird, um das Sitzungsobjekt zu konfigurieren, das vom zurückgegebenen Wiedergabecontroller verwendet wird.
AnzeigenContainer

Die Ansicht, in der die Anzeige angezeigt wird, und die Benutzeroberfläche der Anzeigeninformationen wird gerendert.
BegleiterSlots

Ein Array von zu befüllenden BCOVPulseCompanionSlots.
Strategie

Eine Ansichtsstrategie, die die Ansicht für den zurückgegebenen Wiedergabe-Controller bestimmt.
Optionen

Ein NSDictionary von Pulse-Optionen. Kann null oder leer sein. Die einzigen gültigen Optionsschlüssel sind kBCOVPulseOptionPulsePlaybackSessionDelegateKey kBCOVPulseOptionPulseDeviceContainerKey kBCOVPulseOptionPulsePersistentIdKey

Rückgabewert

Ein neuer Playback-Controller mit den angegebenen Parametern.

Diskussion
