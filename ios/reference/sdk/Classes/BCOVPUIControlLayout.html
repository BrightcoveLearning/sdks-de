BCOVPUIControlLayout Das BCOVPUIControlLayout-Objekt wird mit einer Reihe von eingebetteten Steuerelementen erstellt BCOVPUILayoutView Objekte mit einigen Customizing-Parametern. Das BCOVPUIControlLayout-Objekt wird in a . installiert BCOVPUILayoutView zur Anzeige in einer Spieleransicht.

  StandardLayoutItems

standardLayoutItems ist das Array mit einem oder mehreren Unterarrays von Steuerelementen, die angezeigt werden, wenn die Player-Ansichtsbreite größer als . ist KompaktLayoutMaximale Breite. Jedes Subarray enthält eine Liste von BCOVPUILayoutView Objekte, die in a . präsentiert werden BCOVPUILayoutView. Jedes Subarray repräsentiert eine Kontrollleiste in der Player-Ansicht. standardLayoutItems besteht normalerweise aus einem einzelnen Subarray, das eine einzelne Zeile von Steuerelementen darstellt.

  kompakteLayoutItems

compactLayoutItems ist das Array mit einem oder mehreren Unterarrays von Steuerelementen, die angezeigt werden, wenn die Player-Ansichtsbreite kleiner als . ist KompaktLayoutMaximale Breite. Jedes Subarray enthält eine Liste von BCOVPUILayoutView Objekte, die in a . präsentiert werden BCOVPUILayoutView. Jedes Subarray repräsentiert eine Kontrollleiste in der Player-Ansicht. compactLayoutItems besteht normalerweise aus zwei Unterarrays, die zwei Zeilen von Steuerelementen darstellen, wobei jede Zeile weniger Steuerelemente enthält, um besser an die Breite eines iPhone-Bildschirms anzupassen.

– initWithStandardControls:compactControls:

Initialisiert ein BCOVPUIControlLayout-Layout mit Standardeinstellungen.

- (Instanztyp)initWithStandardControls:(NSArray *)StandardLayoutLines compactControls:(NSArray *)kompakteLayoutLines

Parameter

StandardLayoutLines

Ein Array von Arrays, wobei jedes Unterarray einer Reihe von Steuerelementen in der Player-Steuerelementansicht entspricht. Jedes dieser Unterarrays ist eine Liste von BCOVPUILayoutView-Objekten, die im Layout mit Standardbreite verwendet werden. Jede Layoutansicht enthält typischerweise ein einzelnes Steuerelement und kann mit erstellt werden [BCOVPUIBasicControlView layoutViewWithControlFromTag:width:elasticity:]
kompakteLayoutLines

Ein Array von Arrays, wobei jedes Unterarray einer Reihe von Steuerelementen in der Player-Steuerelementansicht entspricht. Jedes dieser Unterarrays ist eine Liste von BCOVPUILayoutView Objekte, die im kompakten Breitenlayout verwendet werden (normalerweise Hochformat auf einem iPhone).

Rückgabewert

Eine BCOVPUIControlLayout-Instanz, die mit den angegebenen Steuerelementen initialisiert wurde.

Diskussion

Initialisiert ein BCOVPUIControlLayout-Layout mit Standardeinstellungen.
