– addMarkerAt:duration:isAd:color:

Fügen Sie eine Markierung hinzu, die auf dem . angezeigt wird Fortschritt an einem bestimmten Ort ansehen.

- (void)addMarkerAt:(double)Position Dauer: (doppelt)Dauer isAd:(BOOL)isAd Farbe: (UIColor *)Farbe

Parameter

Position

Die Position (in Sekunden) im Fortschritt sehen Sie, wo der Marker platziert werden soll. Siehe die “Dauer” Eigenschaft für verwandte Details.
Dauer

Die Dauer in Sekunden, dargestellt durch die Markierung; kann auf Null gesetzt werden, um die kleinste Markierung (einen Kreis) zu zeichnen.
isAd

JA bedeutet, dass diese Markierung eine Anzeige darstellt. NO zeigt an, dass es sich um einen generischen Marker handelt (standardmäßig weiß gezeichnet). Im Allgemeinen sollten Sie für diesen Wert NO übergeben, da Anzeigenmarkierungen jedes Mal entfernt und neu hinzugefügt werden, wenn Sie zu einer neuen Sitzung wechseln.
Farbe

Eine UIColor, die zum Zeichnen des Markers verwendet wird. Kann auf Null gesetzt werden.

Diskussion

Fügen Sie eine Markierung hinzu, die auf dem . angezeigt wird Fortschritt an einem bestimmten Ort ansehen.
