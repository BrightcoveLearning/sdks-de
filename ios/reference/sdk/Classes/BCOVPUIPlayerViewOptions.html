  panTrägheit

Bestimmt die Geschwindigkeit, mit der ein Schwenk-Gesten-Schleuder abklingt. Standardmäßig ist 1.0. Größere Werte lassen den Schleudergang langsamer abklingen. Kleinere Werte führen dazu, dass der Schleudergang schnell stoppt. Bei einem Wert von 0.0 wird das Schwenken sofort nach dem Touch-Up beendet.

@property (nichtatomar, zuweisen) float panInertia

  automaticControlTypeSelectionUsesShutter

Wenn diese Option aktiviert ist, wird eine Verschlussansicht hinzugefügt und ausgeblendet, wenn die Kontrollansichten zwischen den Videos geändert werden. Wenn dies nicht aktiviert ist, kann es sein, dass Sie beim Wechseln der Kontrollansichten zwischen den Videos ein Flackern sehen. Standardmäßig auf JAWOHL.

@property (nichtatomar, zuweisen) BOOL automaticControlTypeSelectionUsesShutter

Diskussion

Wenn diese Option aktiviert ist, wird eine Verschlussansicht hinzugefügt und ausgeblendet, wenn die Kontrollansichten zwischen den Videos geändert werden. Wenn dies nicht aktiviert ist, kann es sein, dass Sie beim Wechseln der Kontrollansichten zwischen den Videos ein Flackern sehen. Standardmäßig auf JAWOHL.
