Überblick

Kommuniziert das Ergebnis von a BCOVCuePointProgressPolicy Einwände gegen den Kodex, der die Richtlinie konsultiert.

– initWithCuePoints:resumeCuePoint:

Gibt eine initialisierte Instanz von . zurück BCOVCuePointProgressPolicy mit den angegebenen Cue-Punkten und dem Cue-Punkt, an dem die Wiedergabe fortgesetzt werden soll. Geben Sie a . an Null LebenslaufCuePoint um anzuzeigen, dass die Wiedergabe ab der aktuellen Abspielposition fortgesetzt werden soll.

- (Instanztyp)initWithCuePoints:(BCOVCuePointCollection *)cuePoints LebenslaufCuePoint:(BCOVCuePoint *)LebenslaufCuePoint

Parameter

cuePoints

Die Cue-Punkte, die vom Client dieser Richtlinie verarbeitet werden sollten, wie in der Richtlinie festgelegt.
LebenslaufCuePoint

Der Cue-Punkt, dessen Position die Zeit darstellt, zu der die Wiedergabe fortgesetzt werden soll. Angeben Null um anzuzeigen, dass die Wiedergabe ab der aktuellen Abspielposition fortgesetzt werden soll.

Rückgabewert

Ein neu initialisiertes Ergebnis der Cue-Point-Fortschrittsrichtlinie.

  LebenslaufCuePoint

Gibt den Cue-Punkt an, an dem die Wiedergabe fortgesetzt werden soll, sobald der Code die BCOVCuePointProgressPolicy hat die Verarbeitung dieses Ergebnisses abgeschlossen’s Cue-Punkte. Kann null sein (was angibt, dass die Wiedergabe dort fortgesetzt werden soll, wo sich die Abspielposition gerade befindet).
