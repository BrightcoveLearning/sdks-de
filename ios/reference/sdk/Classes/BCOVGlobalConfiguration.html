Überblick

Die globale Konfiguration enthält Eigenschaften, die im gesamten Native Player SDK verwendet werden. Wenn Ihre App eine benutzerdefinierte Konfiguration erfordert, legen Sie die erforderlichen Eigenschaften der globalen Konfiguration fest, BEVOR Sie einen der Dienste des Native Player SDK verwenden.

  chinaProxyDomain

Legt einen benutzerdefinierten, vollqualifizierten Domänennamen (FQDN) für Katalog-API-, Metrik- und Analytics-Sammlungsdienste fest. Diese Methode soll speziell für die zuverlässige Bereitstellung von Inhalten in China verwendet werden und erfordert besondere Kontovereinbarungen. Das Festlegen dieser Eigenschaft in einer anderen Situation führt zu unbekanntem Verhalten.
