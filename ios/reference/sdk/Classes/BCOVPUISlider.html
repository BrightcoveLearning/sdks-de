  Dauer

Die Dauer (in Sekunden), die durch den Bereich des Schiebereglers dargestellt wird. Wird zum Positionieren von Markierungen verwendet. Der Standardwert ist 30, sollte jedoch auf die Länge des zugehörigen Videos eingestellt werden. Die Dauer wird automatisch für den Fortschrittsschieberegler eingestellt, der in der PlayerUI unter playerView.controlsView.progressSlider installiert ist.

– addMarkerAt:duration:isAd:image:

Fügen Sie eine Markierung hinzu, die auf dem Schieberegler an einer bestimmten Stelle angezeigt wird.

- (void)addMarkerAt:(double)Position Dauer: (doppelt)Dauer isAd:(BOOL)isAd Bild:(UIImage *)Bild

Parameter

Position

Die Position (in Sekunden) auf dem Schieberegler, an der der Marker platziert werden soll. Siehe die “Dauer” Eigenschaft für verwandte Details.
Dauer

Die Dauer dargestellt durch das Häkchen; kann auf Null gesetzt werden, um die kleinste Größenmarkierung (2 Pixel breit) zu zeichnen.
isAd

JA bedeutet, dass diese Markierung eine Anzeige darstellt (standardmäßig gelb gezeichnet). NO zeigt an, dass es sich um einen generischen Marker handelt (standardmäßig weiß gezeichnet).
Bild

Ein optionales Bild, das anstelle des standardmäßigen Markierungsindikators gezeichnet wird. Kann auf Null gesetzt werden.

Diskussion

Fügen Sie eine Markierung hinzu, die auf dem Schieberegler an einer bestimmten Stelle angezeigt wird.

– setCustomMinimumTrackImage:forState:

Legen Sie ein benutzerdefiniertes Mindestspurbild fest, das im normalen Modus (ohne Werbung) verwendet werden soll.

- (void)setCustomMinimumTrackImage:(UIImage *)Bild forState:(UIControlState)Zustand

Parameter

Bild

Das Bild, das am unteren Ende der Spur verwendet werden soll (normalerweise die linke Seite des Daumens). Das Bild sollte ein in der Größe veränderbares UIImage sein, ‘trackHeight’ Pixel hoch. Wenn ein benutzerdefiniertes Bild festgelegt ist, werden keine Markierungen mehr angezeigt. Auf nil setzen, um die Standardspurbilder wiederherzustellen.
Zustand

Der UIControlState, für den dieses Bild aktiv ist.

Diskussion

Legen Sie ein benutzerdefiniertes Mindestspurbild fest, das im normalen Modus (ohne Werbung) verwendet werden soll.
