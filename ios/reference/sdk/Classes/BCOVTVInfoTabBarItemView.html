BCOVTVInfoTabBarItemView-Klasse für die “Die Info” Registerkartenleiste Die Standardansicht zeigt grundlegende Informationen aus der Playback-API-Antwort an. Sie können diese Informationen manuell überschreiben, indem Sie die Textbeschriftungen und die Bildansicht einstellen, wenn tabBarItemViewDidAdvanceToPlaybackSession: wird genannt.
