BCOVPlaybackSessionProvider-Protokollreferenz

Entspricht NSObject
Deklariert in BCOVPlaybackSessionProvider.h

Überblick

Objekte, die diesem Protokoll entsprechen, bieten eine Reihe von Wiedergabesitzungen an, die den angegebenen Videos entsprechen.

– WiedergabeSessionsForVideos:

Veraltet: Nicht verwenden Gibt eine Reihe von IDs zurückBCOVPlaybackSession Objekte, die den angegebenen Videos entsprechen. Die erste Wiedergabesitzung wird gesendet, sobald sie verfügbar ist. Jede nachfolgende Wiedergabesitzung wird gesendet, nachdem ihre vorherige (führende) Wiedergabesitzung beendet wurde. Wiedergabesitzungen können asynchron geliefert werden.

- (id)playbackSessionsForVideos:(id<NSFastEnumeration>)Videos

Parameter

Videos

Die BCOVVideo Objekte, denen die zurückgegebenen Wiedergabesitzungsobjekte entsprechen sollen.

Rückgabewert

Die BCOVPlaybackSession Objekte, die den angegebenen Videos entsprechen.

Diskussion

Warnung: Veraltet: Nicht verwenden Gibt eine Reihe von IDs zurückBCOVPlaybackSession Objekte, die den angegebenen Videos entsprechen. Wiedergabesitzungen können asynchron geliefert werden.
