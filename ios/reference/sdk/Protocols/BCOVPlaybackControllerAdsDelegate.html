Entspricht NSObject
Deklariert in BCOVWerbung.h

Überblick

Halten Sie sich an dieses Protokoll, um grundlegende Anzeigeninformationen für jede Sitzung zu erhalten.

– PlaybackController:playbackSession:didEnterAdSequence:

Wird aufgerufen, wenn die Wiedergabe innerhalb einer Wiedergabesitzung in eine neue Anzeigensequenz eintritt.

- (void)playbackController:(id<BCOVPlaybackController>)Regler PlaybackSession:(id<BCOVPlaybackSession>)Sitzung didEnterAdSequence:(BCOVAdSequence *)adSequence

Parameter

Regler

Der Wiedergabe-Controller, in dem dieser Übergang aufgetreten ist.
Sitzung

Die Wiedergabesitzung, in der die Anzeigenumstellung stattfand.
adSequence

Die eingegebene Anzeigensequenz.

Diskussion

Wird aufgerufen, wenn die Wiedergabe innerhalb einer Wiedergabesitzung in eine neue Anzeigensequenz eintritt.

BCOVWerbung.h
