Überblick

Eine spezifische “Quelle” oder Wiedergabe innerhalb von a BCOVVideo Objekt. Während Brightcove Inhalte in Entitäten organisiert, die als Videos bezeichnet werden, werden die tatsächlichen Medien-URLs, über die auf die Inhalte zugegriffen werden kann, durch eine Quelle gekapselt. Neben der URL kann eine Quelle einen optionalen Zustellungstyp haben, der verwendet werden kann, um sie von anderen Quellen zu unterscheiden, und eine Reihe von Eigenschaften anders als die Eigenschaften auf dem Video, das es besitzt.

Beachten Sie, dass ein Video mehrere Quellen mit demselben Übermittlungstyp enthalten darf. In diesem Fall müssen die Quellen auf andere Weise eindeutig gemacht werden (z. B. mit unterschiedlichen Werten in ihren jeweiligen Eigenschaften).

– aktualisieren: erforderliche Methode

Gibt eine geänderte Version dieser Quelle zurück. Da BCOVSource-Objekte unveränderlich sind, muss eine völlig neue BCOVSource erstellt werden, selbst wenn nur eine einzige Änderung erforderlich ist. Daher bietet diese Methode eine bequeme Möglichkeit, eine temporäre veränderbare Kopie dieser Instanz zu erhalten und sie zu ändern (innerhalb von updateBlock).

- (Instanztyp)update:(void ( ^ ) ( id<BCOVMutableSource> mutableSource ))updateBlock

Parameter

updateBlock

Ein Block, dem eine veränderliche Kopie dieser Quelle übergeben wird.

Rückgabewert

Die Kopie dieser Quelle geändert von updateBlock.

Diskussion

Gibt eine geänderte Version dieser Quelle zurück. Da BCOVSource-Objekte unveränderlich sind, muss eine völlig neue BCOVSource erstellt werden, selbst wenn nur eine einzige Änderung erforderlich ist. Daher bietet diese Methode eine bequeme Möglichkeit, eine temporäre veränderbare Kopie dieser Instanz zu erhalten und sie zu ändern (innerhalb von updateBlock).

Versuchen Sie nicht, einen Verweis auf die an . übergebene veränderliche Quelle zu speichern updateBlock , da das Verhalten dabei undefiniert ist. Nehmen Sie stattdessen alle erforderlichen Änderungen im Block vor und erfassen Sie dann das von dieser Methode zurückgegebene Quellobjekt.
