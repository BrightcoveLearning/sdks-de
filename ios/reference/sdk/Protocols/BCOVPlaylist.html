Überblick

Wertklasse, die eine Playlist im Brightcove Player SDK-Ökosystem darstellt. Eine Playlist ist eine geordnete Sammlung von Videos , sowie zusätzliche optionale Metadaten zur Sammlung in Form von Eigenschaften. Beachten Sie, dass BCOVPlaylist, da es NSFastEnumeration implementiert, in Methoden verwendet werden kann, die NSFastEnumeration-Parameter verwenden (wie z [BCOVPlaybackController setVideos:]).

– aktualisieren: erforderliche Methode

Gibt eine geänderte Version dieser Playlist zurück. Da BCOVPlaylist-Objekte unveränderlich sind, muss eine völlig neue BCOVPlaylist erstellt werden, selbst wenn nur eine einzige Änderung erforderlich ist. Daher bietet diese Methode eine bequeme Möglichkeit, eine temporäre veränderbare Kopie dieser Instanz zu erhalten und sie zu ändern (innerhalb von updateBlock).

- (Instanztyp)update:(void ( ^ ) ( id<BCOVMutablePlaylist> mutablePlaylist ))updateBlock

Parameter

updateBlock

Ein Block, dem eine veränderliche Kopie dieser Playlist übergeben wird.

Rückgabewert

Die Kopie dieses Cue-Points geändert von updateBlock.

Diskussion

Gibt eine geänderte Version dieser Playlist zurück. Da BCOVPlaylist-Objekte unveränderlich sind, muss eine völlig neue BCOVPlaylist erstellt werden, selbst wenn nur eine einzige Änderung erforderlich ist. Daher bietet diese Methode eine bequeme Möglichkeit, eine temporäre veränderbare Kopie dieser Instanz zu erhalten und sie zu ändern (innerhalb von updateBlock).

Versuchen Sie nicht, einen Verweis auf die an . übergebene veränderbare Wiedergabeliste zu speichern updateBlock , da das Verhalten dabei undefiniert ist. Nehmen Sie stattdessen alle erforderlichen Änderungen im Block vor und erfassen Sie dann das von dieser Methode zurückgegebene Wiedergabelistenobjekt.
