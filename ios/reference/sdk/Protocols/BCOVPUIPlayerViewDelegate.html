– SpielerViewShouldDisplayThumbnailPreviewWithRect:

Wird kurz vor dem Einrichten der Miniaturansicht aufgerufen.

- (CGRect)playerViewShouldDisplayThumbnailPreviewWithRect:(BCOVPUIPlayerView *)SpielerView

Parameter

SpielerView

Die Player-Ansicht, in der die Miniaturansicht angezeigt wird.

Rückgabewert

Das angepasste CGRect für die Miniaturansicht

Diskussion

Wird kurz vor dem Einrichten der Miniaturansicht aufgerufen.

Sie können dies implementieren, um die Miniaturansichtsvorschaugröße für verschiedene Geräte, Layouts, Bildschirmmodi usw. anzupassen.

Die x Der Wert von CGRect wird ignoriert, sodass Sie die x Wert auf 0 gesetzt.

Wenn diese Delegate-Methode nicht implementiert ist, beträgt die Größe der Miniaturansicht 100 x 56 mit einem vertikalen Versatz von -60.
