BCOVPlaybackServiceErrorCode

Fehler im Zusammenhang mit Playback API-Anfragen.

Definition

typedef NS_ENUM(NSInteger, BCOVPlaybackServiceErrorCode ) {
   BCOVPlaybackServiceErrorNoError,
   BCOVPlaybackServiceErrorCodeConnectionError,
   BCOVPlaybackServiceErrorCodeJSONDeserializationError,
   BCOVPlaybackServiceErrorCodeAPIError,
};

Konstanten

BCOVPlaybackServiceErrorNoError

Kein Fehler.

Deklariert in BCOVPlaybackService.h.

BCOVPlaybackServiceErrorCodeConnectionError

Fehlercode, der angibt, dass bei der Verbindung mit der Playback-API ein Fehler aufgetreten ist. Überprüfen Sie den Schlüssel error.userInfo NSUnderlyingErrorKey auf den tatsächlichen Verbindungsfehler.

Deklariert in BCOVPlaybackService.h.

BCOVPlaybackServiceErrorCodeJSONDeserializationError

Fehlercode, der angibt, dass beim Parsen der Antwort als JSON ein Fehler aufgetreten ist. Wenn die Anforderung Daten zurückgegeben hat, werden diese rohen NSData durch den Schlüssel kBCOVPlaybackServiceErrorKeyRawResponseData in die userInfo aufgenommen. Überprüfen Sie den Schlüssel error.userInfo NSUnderlyingErrorKey auf den tatsächlichen JSON-Deserialisierungsfehler.

Deklariert in BCOVPlaybackService.h.

BCOVPlaybackServiceErrorCodeAPIError

Fehlercode, der angibt, dass von der API ein Fehler zurückgegeben wurde. Überprüfen Sie den Schlüssel error.userInfo kBCOVPlaybackServiceErrorKeyAPIErrors auf ein NSArray von Fehlern, die von der API zurückgegeben wurden, und den Schlüssel kBCOVPlaybackServiceErrorKeyAPIHTTPStatusCode, um den zurückgegebenen Statuscode zu finden.
